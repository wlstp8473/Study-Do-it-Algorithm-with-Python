# 파이썬의 리스트와 튜플

## 1. 리스트(List)

리스트는 여러 원소를 순서대로 저장하는 **변경 가능한(Mutable) 자료형**이다.

### 리스트 생성

대괄호 `[]` 또는 `list()`를 사용하여 생성한다.

```python
list_01 = ['A', 'B', 'C']
list_02 = list('ABC')

print(list_01)  # ['A', 'B', 'C']
print(list_02)  # ['A', 'B', 'C']
```

`list('ABC')`는 문자열의 각 문자를 원소로 사용한다.

### 초깃값이 정해지지 않은 리스트

`None`을 사용하면 값이 아직 정해지지 않은 리스트를 만들 수 있다.

```python
list_03 = [None] * 5

print(list_03)
# [None, None, None, None, None]
```

리스트는 생성 후에도 `append()`와 같은 메서드로 원소를 추가할 수 있으므로 원소 개수를 미리 확정할 필요는 없다.

```python
numbers = []
numbers.append(10)
numbers.append(20)

print(numbers)  # [10, 20]
```

---

## 2. 튜플(Tuple)

튜플은 여러 원소를 순서대로 저장하지만, 생성한 뒤에는 원소를 변경할 수 없는 **불변(Immutable) 자료형**이다.

### 튜플 생성

소괄호 `()` 또는 `tuple()`을 사용한다. 소괄호는 생략할 수도 있다.

```python
tuple_01 = (1, 2, 3)
tuple_02 = 1, 2, 3
tuple_03 = tuple([1, 2, 3])
```

### 원소가 1개인 튜플

원소가 하나인 튜플을 만들 때는 반드시 쉼표 `,`를 붙인다.

```python
tuple_01 = (1,)
tuple_02 = 1,
```

쉼표가 없으면 튜플이 아니라 일반 정수로 처리된다.

```python
value = (1)

print(type(value))  # <class 'int'>
```

### 핵심 차이

```python
numbers_list = [1, 2, 3]
numbers_list[0] = 10
```

```python
numbers_tuple = (1, 2, 3)
```

### 튜플의 `+=`, `*=`

튜플에도 `+=`와 `*=`를 사용할 수 있다. 하지만 기존 객체를 변경하는 것이 아니라 새로운 튜플을 생성하여 변수에 다시 대입한다.

```python
numbers = (1, 2)
before_id = id(numbers)

numbers += (3,)
after_id = id(numbers)

print(numbers)                # (1, 2, 3)
print(before_id == after_id)  # False
```

```python
}
```

# 파이썬 자료형의 분류

## 변경 가능한 자료형(Mutable)

객체가 생성된 후 내부 값을 변경할 수 있다.

- 리스트(`list`)
- 딕셔너리(`dict`)
- 집합(`set`)

## 변경 불가능한 자료형(Immutable)

객체가 생성된 후 내부 값을 변경할 수 없다.

- 정수(`int`)
- 실수(`float`)
- 문자열(`str`)
- 튜플(`tuple`)
- 불리언(`bool`)

```python
number = 10
number = 20
```

위 코드에서는 정수 객체 `10`이 `20`으로 변경된 것이 아니다. 변수 `number`가 새로운 정수 객체 `20`을 참조하게 된 것이다.

# 파이썬에서 대입의 의미

파이썬의 변수는 객체 자체를 담는 상자라기보다 **객체를 참조하는 이름**으로 이해할 수 있다.

```python
number = 10
```

```text
number ──참조──▶ 정수 객체 10
```

다른 값을 대입하면 변수가 참조하는 객체가 바뀐다.

```python
number = 20
```

```text
number ──참조──▶ 정수 객체 20
```

객체의 식별값은 `id()`로 확인할 수 있다.

```python
number = 10
print(id(number))

number = 20
print(id(number))
```

## 대입의 특징

1. 변수에 처음 값을 대입하면 해당 이름이 생성된다.
2. 변수에는 객체에 대한 참조가 대입된다.
3. 변수에 다른 자료형의 객체를 다시 대입할 수 있다.
4. 여러 변수에 여러 값을 한 번에 대입할 수 있다.
5. 파이썬의 `=`는 대입문에 사용하는 기호이다.

### 동적 타이핑

파이썬에서는 변수의 자료형을 미리 선언하지 않는다. 자료형은 변수가 아니라 변수가 참조하는 객체에 속한다.

```python
value = 10        # int 객체 참조
value = 'Python'  # str 객체 참조
```

### 여러 값 대입과 언패킹

```python
a, b, c = 1, 2, 3

print(a)  # 1
print(b)  # 2
print(c)  # 3
```

두 변수의 값을 간단하게 교환할 수도 있다.

```python
a, b = b, a
```

# 슬라이스(Slice)

슬라이스는 리스트나 튜플 같은 시퀀스에서 일정 범위의 원소를 가져오는 기능이다.

## 기본 형식

```python
s[i:j]
```

인덱스 `i`부터 `j - 1`까지 가져온다. 끝 인덱스 `j`는 포함하지 않는다.

```python
s[i:j:k]
```

인덱스 `i`부터 `j - 1`까지 `k` 간격으로 가져온다.

- `i`: 시작 인덱스
- `j`: 종료 인덱스
- `k`: 이동 간격(step)

## 슬라이스의 특징

1. 시작 또는 종료 인덱스가 범위를 벗어나도 일반적으로 오류가 발생하지 않는다.
2. 양의 간격에서 시작값을 생략하면 처음부터 가져온다.
3. 양의 간격에서 종료값을 생략하면 끝까지 가져온다.
4. 간격을 생략하면 기본값은 `1`이다.
5. 간격에는 음수를 사용할 수 있다.
6. 간격에 `0`을 사용하면 `ValueError`가 발생한다.

```python
numbers = [11, 22, 33]

print(numbers[:100])  # [11, 22, 33]
print(numbers[100:])  # []
```

> 음수 간격을 사용하면 시작값과 종료값의 기본 방향도 반대로 적용된다.

## 슬라이스 사용 예

다음 리스트를 기준으로 살펴본다.

```python
s = [11, 22, 33, 44, 55, 66, 77]
```

| 패턴 | 설명 | 실행 결과 |
|---|---|---|
| `s[:]` | 모든 원소를 가져온다. | `[11, 22, 33, 44, 55, 66, 77]` |
| `s[:n]` | 처음부터 `n`개를 가져온다. | `s[:3]` → `[11, 22, 33]` |
| `s[i:]` | `s[i]`부터 마지막까지 가져온다. | `s[3:]` → `[44, 55, 66, 77]` |
| `s[-n:]` | 뒤에서 `n`개의 원소를 가져온다. | `s[-3:]` → `[55, 66, 77]` |
| `s[::k]` | 처음부터 `k`칸 간격으로 가져온다. | `s[::2]` → `[11, 33, 55, 77]` |
| `s[::-1]` | 모든 원소를 역순으로 가져온다. | `[77, 66, 55, 44, 33, 22, 11]` |

## 인덱싱과 슬라이싱의 차이

인덱싱은 범위를 벗어나면 `IndexError`가 발생한다.

```python
s[100]  # IndexError
```

슬라이싱은 범위를 벗어나도 가능한 범위까지만 처리한다.

```python
s[:100]  # [11, 22, 33, 44, 55, 66, 77]
```
